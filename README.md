# BEX - AI Assistant Prototype

version 0.6.0

BEX is a C# console-based AI assistant powered by Gemini API.

## Features

- AI conversation through Gemini API
- JSON-based persistent memory system
- Automatic note saving
- Async API communication
- Console-based user interaction

## Technologies

- C#
- .NET 8
- Google Gemini API
- System.Text.Json
