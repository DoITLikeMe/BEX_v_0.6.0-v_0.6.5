# BEX - AI Assistant Prototype

version 0.6.0

BEX is a C# console-based AI assistant powered by Gemini API.

## Features

- AI conversation through Gemini API
- JSON-based persistent memory system
- Automatic note saving
- Async API communication
- Console-based user interaction

## Technologies

- C#
- .NET 8
- Google Gemini API
- System.Text.Json

## Current version

v0.6.0


Set `GEMINI_API_KEY` environment variable before running.

## How to run

1. Clone the repository

2. Set your Gemini API key:
  - GEMINI_API_KEY= your_api_key_here

3. build and run the project.
